# Learn Sequelize

A learning repository for practicing Sequelize ORM with Node.js, covering database models, migrations, and CRUD operations against a relational database.

## About

This repo documents a step-by-step journey through learning Sequelize, organized into chapters and focused practice modules. It is intended as a personal reference and practice ground for working with Sequelize in Node.js applications.

## Project Structure

```
Learn-Sequelize/
├── CRUD/         # Create, Read, Update, Delete examples using Sequelize
├── chapter 1/    # Foundational Sequelize concepts and exercises
└── README.md
```

## Tech Stack

- Node.js
- Sequelize - Promise-based ORM for Node.js
- SQL Database (e.g., PostgreSQL / MySQL / SQLite)

## Topics Covered

- Setting up Sequelize with a Node.js project
- Defining models and schemas
- Database migrations and seeders
- Performing CRUD operations
- Associations between models

## Getting Started

Clone the repository:

```bash
git clone https://github.com/AryanHulawale/Learn-Sequelize.git
cd Learn-Sequelize
```

Install dependencies (inside the relevant module folder, e.g. `CRUD`):

```bash
npm install
```

Configure your database connection (update Sequelize config with your DB credentials), then run:

```bash
node index.js
```
